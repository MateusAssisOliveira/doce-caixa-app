# **App Name**: Doce Caixa

## Core Features:

- Product Catalog: Display products with filters.
- Shopping Cart: Enable users to add, remove, and update items in a shopping cart.
- Checkout Process: Implement a three-step checkout process.
- Order History: Display the order history for each customer.
- Favorites System: Allow users to save their favorite products for easy access.
- Dashboard Metrics: Generate real time financial metrics, and projections.
- Cash Control: Enable cashiers to control opening/closing balances, log income/expense with categories

## Style Guidelines:

- Primary color: Light, soft pink (#F9E4D4) evoking a feeling of sweetness and comfort.
- Background color: Very light cream (#F5F5DC) offering a clean, inviting backdrop.
- Accent color: Soft lavender (#E6E6FA) for a touch of elegance.
- Headline font: 'Playfair', serif. Body font: 'PT Sans', sans-serif. Note: currently only Google Fonts are supported.
- Use custom icons to represent the product categories.
- Maintain a clean, responsive design.
- Use subtle animations.
